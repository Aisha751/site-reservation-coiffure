🚀 Comment Démarrer

Option 1 : Ouverture Directe (Plus Simple)

1️⃣ Ouvrir le site

Double-cliquez simplement sur index.html dans le dossier du projet

Le site s'ouvrira dans votre navigateur préféré !

Option 2 : Avec Python (Recommandé)

2️⃣ Python 3

python -m http.server 8000

Puis ouvrez: http://localhost:8000

2️⃣ Python 2

python -m SimpleHTTPServer 8000

Puis ouvrez: http://localhost:8000

Option 3 : Avec Node.js

3️⃣ Démarrer le serveur Express

npm install npm start

Puis ouvrez: http://localhost:3000

Option 4 : Avec PHP

4️⃣ Serveur PHP

php -S localhost:8000

Puis ouvrez: http://localhost:8000

❓ Questions Fréquentes

Le site ne s'ouvre pas ?

✓ Vérifiez que tous les fichiers sont dans le même dossier

✓ Essayez avec Python ou Node.js au lieu de double-cliquer

Mes réservations disparaissent ?

✓ Les données sont stockées localement (localStorage)

✓ Effacez le cache du navigateur ou utilisez une fenêtre privée

Comment intégrer une vrai API de paiement ?

Consultez les documentations de Stripe, PayPal, Mollie, etc.

Modifiez la fonction processPayment() dans script.js
